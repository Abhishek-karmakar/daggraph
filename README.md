## Install

```sh
yarn add dgraph
```

## Usage

```
Usage

   $ dgraph <command> <params>

   $ dgraph sample <param>             # Uses the <PARAM>
   
 Examples

   $ dgraph sample TEST                # Uses the TEST
   $ dgraph sample YOLO                # Uses the YOLO
```
